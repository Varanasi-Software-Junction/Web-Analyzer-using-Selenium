PP   Tkinter Stock Trading GUI

Files
- function.py : stock analysis + demo trading logic + JSON persistence
- app.py      :   Tkinter GUI with table view and chart preview
- requirements.txt

Run
1. pip install -r requirements.txt
2. python app.py

Features
- Analyze one share
- Compare two shares
- Save raw data and stats to CSV and JSON
- Save charts as PNG
- Preview charts inside the GUI
- Table view for statistics
- Demo user creation, deposit, buy, sell
- Portfolio holdings table
- Transactions table
- Market snapshot saving
